#前端传递数组问题

##获取前端传递的数组
可以使用json_decode的getContent方法获得，在从中取得相应的值。
、、、
*
$a = json_decode($request -> getContent());
$group_name = $a -> group_name;
$members = $a -> members;
、、、

##使用foreach把取得的值插入数据库
、、、
*
$sql = ‘insert into chat_group (member_id,group_name,member_level, creat_time) VALUES’;
foreach($members as $member) {
$create_time = date(‘Y-m-d H:i:s’,time());
$mem_id = “‘“.$member->id.”‘“;
$g_name = “‘“.$group_name.”‘“;
$c_time = “‘“.$create_time.”‘“;

    $sql .= '(' . $mem_id.','.$g_name.','.$member->level.','.$c_time.')'.',';
  }
$sql = rtrim($sql, ',');
 DB::insert($sql);
return response()->json($sql);
